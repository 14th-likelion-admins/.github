# 멋쟁이사자처럼 광운대학교 14기

광운대학교 멋쟁이사자처럼 14기 운영진이 직접 기획·디자인·개발한 공식 홈페이지입니다.[web:2][web:6]  
“아이디어를 서비스로, 가능성을 현실로”라는 슬로건 아래, 광운대 아기사자들의 성장을 기록하고 연결하는 공간을 목표로 합니다.[web:5][web:7][web:10]

---

## 🦁 About 멋쟁이사자처럼 광운대학교

- 멋쟁이사자처럼은 전국 대학에서 활동하는 IT·AI 창업 동아리로, 아이디어를 실제 서비스로 구현하는 것을 지향합니다.[web:5][web:7][web:10]  
- 광운대학교 지부는 웹/앱 개발, 해커톤, 아이디어톤 등 다양한 활동을 통해 개발 경험이 적은 학생도 서비스 런칭까지 경험할 수 있도록 돕고 있습니다.[web:6][web:7]

---

## 🌐 이 프로젝트는

- **팀**: 멋쟁이사자처럼 광운대학교 14기 운영진  
- **성격**: 14기 첫 공식 프로젝트이자, 멋쟁이사자처럼 광운대학교의 프로젝트 아카이브 & 리크루팅 허브  
- **목표**  
  - 기수별·팀별 프로젝트를 한 곳에서 모아 볼 수 있는 아카이브 제공  
  - 지원자를 위한 활동 소개 및 지원 동선 제공[web:2][web:6]  
  - 운영진·아기사자 모두에게 “우리가 만들어 온 결과물”을 확인할 수 있는 기록 공간 제공[web:6][web:7]

## 🧩 기술 구조

이 레포지토리는 해당 홈페이지의 **Frontend** 코드가 담긴 저장소입니다.[web:2]  
Backend 레포지토리에서는 다음과 같은 기능을 중심으로 서버를 구현하고 있습니다.

- 프로젝트 아카이브 관련 API 설계 및 구현  
- 운영진(Admin) 전용 프로젝트 CRUD 기능  
- 추후 기수/태그/역할별 필터링 및 검색 기능 확장 예정[web:6][web:7]
